# Xilinx Vitis IDE Codes Description
1. local_run_testing_radar_processing.c : This file is for running radra simulation locally on your own device.
2. radar_processing_timming_commented.c : This file is for running on the PS of the zedboard via Vitis IDE.
3. radar_processing_timing_with_LUT_twiddle_factors.c : This file does radar signal processing in an optimized manner. It uses Look-up Table (LUT) for twiidle factors while doing FFT and IFFT. (for running on zedboard via Vitis IDE)
